### Download

1. Clone o projeto

```shellscript
$ git clone https://github.com/robertosxav/processo-seletivo.git
$ cd esocial
```

### Execução

2. Inicialize a aplicação com `docker-compose`

```shellscript
docker-compose up
```

### Api

3. Acesse  `Swagger `

```shellscript
`http://localhost:8080/swagger-ui/index.html
```
